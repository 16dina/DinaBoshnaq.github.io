ASP.NET Web App with Contentful Headless CMS
Cooking Blog

This project was part of my DevOps and Security class at Thomas More. In this project, I worked with the C# language to make a ASP.NET web app about cooking tutorials. I used a headless CMS called Contentful to store and retrieve teh data of the blog.

What I learned during this project

Through this project, I learned to work with Contentful which has become in demand in many job vacancies nowadays. I also got a basic introduction to C# which was still a fairly new programming language for me. It required much self-search online to look at different documentation for how to use Contentful in order to retrieve the data. This has enhanced my search skills and driven my curiosity to learn more about Contentful. I also learned how to apply the basic principles of DevOps by creating a pipeline to automate the process of building the web app. I enhanced my GitHub skills by using GitHub Actions, and my Docker skills by using Docker Hub to make my own image and container to host the web app.
